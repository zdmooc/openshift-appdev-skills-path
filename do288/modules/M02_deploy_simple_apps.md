
# M02 — Déployer des applications simples (console + oc)

## Objectifs
- Déployer depuis une image
- Déployer depuis un dépôt Git (si tu as un exemple)
- Exposer via Service/Route
- Scale + rollout/rollback

## Pattern “release contrôlée”
1) déployer v1
2) valider route
3) déployer v2
4) surveiller rollout
5) rollback si nécessaire
