
# Red Hat OpenShift Application Developer — Skills Path (DO188 → EX188 → DO288 → EX288)

Ce dépôt est un **programme ultra détaillé, orienté pratique**, pour suivre un chemin de compétences complet vers :
- **DO188** (Containers/Podman + bases OpenShift)
- **EX188** (Red Hat Certified Specialist in Containers)
- **DO288** (Développement & déploiement cloud-native sur OpenShift)
- **EX288** (Red Hat Certified OpenShift Application Developer)

## Comment l’utiliser (mode “examen”)
1. Commencer par **docs/00-START-HERE.md** (préparation poste + cluster).
2. Suivre les modules DO188 + labs, puis basculer en entraînement EX188 (sujets blancs + checklists).
3. Enchaîner DO288 (modules + labs + capstone), puis entraînement EX288 (sujets blancs + runbooks).
4. Dans chaque lab : **ne pas avancer sans “Proof”** (commandes de vérification + résultats attendus).
