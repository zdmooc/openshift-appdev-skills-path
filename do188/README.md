
# DO188 — Plan de travail (ultra détaillé)

## Objectif du track
À la fin, tu dois pouvoir :
- containeriser une app, la builder proprement, la publier, la lancer
- diagnostiquer un container qui ne démarre pas (ports, env, volumes, permissions)
- orchestrer une app simple sur OpenShift (Deployment/Service/Route)
- écrire des preuves (Proof) et nettoyer proprement

## Modules
- M01 : Containers fundamentals
- M02 : Podman basics (run/exec/logs/inspect)
- M03 : Images & registries (tags/digests/auth)
- M04 : Containerfile (build reproductible + bonnes pratiques)
- M05 : Persistance (volumes, bind mounts, permissions)
- M06 : Troubleshooting & remote debug
- M07 : Compose (multi-container local)
- M08 : Orchestration (k8s / OpenShift basics)
